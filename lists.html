Greeks of Greeks - HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

What is HTML?

HTML is a markup language that is used to create the structure and content of web pages. It is made up of a series of elements, which are represented by tags.

Basic HTML Elements

  • Headings: h1, h2, h3, h4, h5, h6
  • Paragraphs: p
  • Links: a
  • Images: img
  • Lists: ul, ol, li

HTML Document Structure

An HTML document typically consists of the following elements:

  1. DOCTYPE declaration: !DOCTYPE html
  2. html element: html
  3. head element: head
  4. body element: body

HTML Formatting Tags

HTML provides several formatting tags that can be used to change the appearance of text:

  • Bold: b
  • Italic: i
  • Underline: u
  • Strikethrough: strike
HTML Elements:
Block-level elements: Take up the full width of their parent element and start on a new line. Examples: div, p, h1, ul, ol, dl. - Inline elements: Take up only the space needed to display their content and do not start on a new line. Examples: span, a, img, input.
HTML Attributes:
Global attributes: Can be applied to any HTML element. Examples: id, class, style, title. - Event attributes: Allow you to attach JavaScript code to specific events. Examples: onclick, onmouseover, onmouseout.
HTML Semantics:
Semantic elements: Provide meaning to the structure of a web page. Examples: header, nav, main, section, article, aside, footer. - Non-semantic elements: Do not provide meaning to the structure of a web page. Examples: div, span.
